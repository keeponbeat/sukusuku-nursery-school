# 要件定義書 (01_requirements.md) - 瀬谷すくすく保育園

## 1. プロジェクト概要
- **プロジェクト名**: 瀬谷すくすく保育園 ウェブサイトリニューアル
- **運営**: 株式会社すくすく（代表：村上 学）
- **URL**: https://suku2.jp (WordPress)
- **サイトの性格**: 横浜市小規模保育事業（定員12名）としての地域密着型・高密度な保育の可視化。

## 2. 目的とターゲット
### 2.1 目的
1. **安心感の醸成**: 入園検討中・決定済みの保護者が「ここに預けて良かった」と確信できる透明性の高い情報発信。
2. **事務負担の軽減**: 登園届のダウンロードや詳細なFAQを整備し、園と保護者双方の窓口業務を効率化する。
3. **採用ブランディング**: 11名のスタッフ全員の専門性を紹介し、質の高い保育士を惹きつける「チームすくすく」の魅力を伝える。

### 2.2 ターゲット
- **メインターゲット**: 横浜市の利用調整を経て当園への入園を検討中、あるいは決定した「初めての育児に不安を抱える」保護者様。
- **サブターゲット**: 就職・転職を検討している保育士、栄養士、調理職。

## 3. サイト構成（サイトマップ）
1. **ホーム (TOP)**
   - メインビジュアル（スライド10枚：プロ撮影 ＋ Amazon Photos厳選日常写真）
   - 教育理念・運営方針のハイライト（園の魂となる4項目を視覚的に表現）
   - 最新ニュース / 重要なお知らせ
   - SNS（Instagram）フィード埋め込み
   - クイックアクセス（見学案内 / 書類ダウンロード / 行政リンク）
   - ※「開園告知」等の期間限定テキストは排除し、ブランドイメージを優先。
2. **園について (About)**
   - **教育理念・運営方針（園の魂）**:
     1. 子どもたちの感性を大切にし、すくすく成長できる環境を整えます。
     2. 子どもたちが「どうぞ」や「ありがとう」を素直に言えるやさしい心を育みます。
     3. 子どもたちが自然に親しみ、自然の中で遊びを楽しみます。
     4. 子どもたちの食に対する興味を育て、丈夫で健康な体をつくります。
   - 管理者・園長挨拶
   - 施設概要（所在地、定員、開園年月日：令和6年4月1日、**連携施設情報** 等）
   - **安全・衛生・健康管理**:
     - 感染症対策の指針
     - 各種訓練（避難訓練、不審者対策等）の年間計画
     - 定期健康診断・歯科検診の実施計画
   - 1日の流れ
   - **年間行事（インタラクティブ・カレンダー）**:
     - 年間の主要行事、検診、訓練を月別に表示。
     - マウスオーバーで写真や詳細説明が浮かび上がるUIを採用。
3. **スタッフ紹介 (Staff Interview) ★目玉企画**
   - 「12人の子どもを、11人のプロが見守る」チーム紹介ページ
   - 11名全員の個別インタビューとポートレート
4. **入園・見学のご案内 (Admission)**
   - 園見学詳細（平日15:30〜の意義：午睡明けの活動が見られる時間）
   - 募集状況の案内（横浜市瀬谷区福祉保健センターへの誘導リンク）
   - 入園までの流れ
5. **保護者の方へ (For Parents)**
   - 提出書類ダウンロード（登園届：保護者用 / 医師用）
   - よくある質問 (FAQ)：おむつ、離乳食、病児対応、卒園後の進路など
   - 園生活のしおり（簡易版）
6. **ブログ / ギャラリー (Daily Log)**
   - Amazon Photosを活用した日常の活動レポート
7. **お問い合わせ (Contact)**
   - フォーム、電話番号、緊急時連絡先

## 4. コンテンツ・素材要件
### 4.1 写真素材
- **プロ撮影**: スタッフ全員分（11名）、集合写真、施設内外観、主要な保育風景。
- **日常アーカイブ**: Amazon Photos上の過去資産から厳選（行事、食事、ふとした表情）。
- **プライバシー保護**: 全園児を対象とした「写真使用同意書」の取得（別途フォーマット作成）。

### 4.2 テキスト・ファイル素材
- **インタビュー原稿**: 11名分（職種別深掘り項目を含む）。
- **FAQ原稿**: 頻出問い合わせの整理と回答（園長ヒアリングに基づく）。
- **ダウンロード書類**: 最新版PDF（登園届 2種）。

## 5. 技術・運用要件
- **プラットフォーム**: 既存の WordPress を活用（要リニューアルに伴う管理画面の整理）。
- **モバイル対応**: 保護者のライフスタイルに合わせ、スマートフォンでの閲覧・PDF確認を最優先したレスポンシブデザイン。
- **運用体制**: インスタグラム・ブログの更新（先生方と相談の上、負担の少ない投稿フローを構築）。
- **写真運用自動化 (Sukusuku Picture Proposer 連携)**:
  - 本機能は独立したソフトウェアプロジェクト [Sukusuku Picture Proposer](../../software/sukusuku-picture-proposer/docs/01_requirements.md) が担当。
  - WordPress等の管理画面、または専用ツール上で AI が「加工済み写真素材（マスキング・切り抜き済み）」を**提案**する。
  - 運用者はその中から素材を選択し、自ら文章を作成して投稿するフローとする。
  - **主な連携機能**: iCloud/Amazon Photos からの素材同期、AI顔認識による自動マスキング、自動切り抜き・構図最適化。

# 写真運用自動化の外部プロジェクト化について

**本ドキュメントの最新情報は、以下のソフトウェアプロジェクトに集約されました。**

[Sukusuku Picture Proposer (ソフトウェアプロジェクト) 要件定義書](../../software/sukusuku-picture-proposer/docs/01_requirements.md)

### 概要
ウェブサイトのリニューアル要件に含まれていた「写真の自動マスキング・キャプション生成」機能は、運用の汎用性と技術的な独立性を考慮し、専用の Web アプリケーションとして新規構築することに決定いたしました。

- **連携の仕組み**: 
  - `Sukusuku Picture Proposer` アプリケーション側で、iCloud/Amazon Photos からの素材抽出、AI 解析、プライバシー加工（顔隠し）、キャプションの自動提案を行います。
  - 加工済みの最終素材と、提案された文言を本ウェブプロジェクト（WordPress等）へ引き渡す形で運用します。

# SNS・ブログ運用仕様書（叩き台）

## 1. Instagram 運用仕様
### 1.1 ブランディング・ディレクション
- **トーン＆マナー**: 「ぬくもり」「発見」「安心」。
- **カラーパレット**: 園のロゴに使用されているパステルカラー（緑、オレンジ、黄色）を基調とする。
- **写真のスタイル**: 
    - 子どもの自然な表情（笑顔だけでなく、真剣な眼差し、驚きなど）。
    - 季節の草花や、散歩中の小さな発見。
    - 手作りの温かさが伝わる給食や制作物。
- **ハッシュタグ**: #瀬谷すくすく保育園 #横浜市瀬谷区 #小規模保育 #食育 #自然保育

### 1.2 投稿頻度・スケジュール
- **頻度**: 週3回（火・木・土を基本とする）。
- **内容例**:
    - 火曜日：お散歩の風景（自然との触れ合い）。
    - 木曜日：給食・食育（今日のおいしい！）。
    - 土曜日：今週のまとめ、または制作作品の紹介。

## 2. ブログ 運用仕様
### 2.1 目的とターゲット
- **目的**: 園の想いや、行事の裏側にある「意図」を深く伝える。
- **ターゲット**: 入園を検討している保護者、在園児の保護者。

### 2.2 投稿頻度
- **頻度**: 月1〜2回。
- **内容例**:
    - 季節の行事（夏祭り、運動会等）の振り返りと、子どもたちの成長。
    - 園長・社長による「保育・経営への想い」。
    - 栄養士による「旬の食材と献立のこだわり」。

## 3. 運用フロー（再掲）
- 撮影：園貸与の端末を使用。
- チェック：投稿前に園長または管理者の確認を必須とする。
- 保存：Amazon Photos等のクラウドストレージにバックアップ。

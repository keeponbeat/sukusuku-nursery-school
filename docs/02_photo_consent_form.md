# 写真・動画使用に関する同意書（フォーマット案）

**瀬谷すくすく保育園 御中**

瀬谷すくすく保育園（以下「当園」）では、園での活動内容を保護者の皆様に共有し、また地域の方々に当園の保育方針を正しくご理解いただくために、公式ウェブサイト、SNS（Instagram等）、およびパンフレット等の広報媒体において、園児の活動風景（写真・動画）を活用することがございます。

つきましては、園児のプライバシー保護を第一に考え、以下の項目に記載の内容をご確認の上、同意いただける範囲をご指定ください。

---

### 【1. 主な使用目的】
- 当園公式ウェブサイト（suku2.jp）での活動紹介。
- 当園公式SNS（Instagram / ブログ）での日常風景の発信。
- 園内掲示物、および入園案内パンフレットへの掲載。

### 【2. プライバシーおよび安全への配慮】
- 掲載にあたっては、原則として園児のフルネームは記載いたしません。
- 営利目的の二次利用や、第三者への提供は一切行いません。
- 同意いただけない場合、または個別に配慮が必要な場合は、顔が判別できないようスタンプ加工やトリミング等の措置を講じます。

---

### 【3. 同意確認項目】
以下のいずれかにチェック（レ）をお願いいたします。

[ ] **1. 全面同意する**
- ウェブサイト、SNS、印刷物等、すべての広報媒体において写真・動画を掲載することに同意します。

[ ] **2. 条件付きで同意する（以下の範囲に限定する）**
- [ ] ウェブサイト（suku2.jp）のみ可
- [ ] SNS（Instagram / ブログ）のみ可
- [ ] 園内掲示物・印刷物のみ可
- [ ] **加工が必要**（顔がはっきり写る場合は、スタンプ等で隠してほしい）

[ ] **3. 全面的に同意しない**
- いかなる媒体においても、写真・動画の掲載を希望しません。
